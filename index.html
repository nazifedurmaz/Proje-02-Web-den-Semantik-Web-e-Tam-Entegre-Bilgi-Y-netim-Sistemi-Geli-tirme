🔗 RDF Turtle Örneği

# Kitap kaynağı libr:book/B001 a lib:Book ; dc:title "Tutunamayanlar" ; lib:hasAuthor libr:author/A001 ; lib:rating 4.8 ; lib:isAvailable true . # Yazar kaynağı libr:author/A001 a lib:Author ; schema:name "Oğuz Atay" ; lib:nationality "Turkish" .

📐 Ontoloji Sınıfları

# Sınıf tanımları lib:Book a owl:Class ; rdfs:subClassOf schema:Book . lib:Author a owl:Class ; rdfs:subClassOf schema:Person . # Özellik tanımları lib:hasAuthor a owl:ObjectProperty ; rdfs:domain lib:Book ; rdfs:range lib:Author .
